# post-upgrade-snapshot-generator

```
Available Commands:
  chain-init              Initialize the chain
  chain-snapshot-export   Export the chain snapshot
  completion              Generate the autocompletion script for the specified shell
  create-second-validator Create a second validator
  delete-snapshot         Delete an existing snapshot file from R2 bucket
  help                    Help about any command
  prepare-validator-data  Prepare validator data
  submit-new-proposal     Submit a new proposal
  upgrade-to-new-binary   Upgrade to the new binary
  upload-snapshot         Upload snapshot file to R2
  version                 Show version
```
